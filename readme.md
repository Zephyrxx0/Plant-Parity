# Plant Parity

A Minecraft mod for **Fabric** that provides compatibility between **Cobblemon** and **Farmer's Delight**.

## Features

- **Crop Compatibility**: Adds Farmer's Delight Rich Soil Farmland to the `hearty_grains_land_plantable` tag, allowing Cobblemon grains to be planted on improved soil.

## Requirements

- **Minecraft**: 1.21.1
- **Fabric Loader**: >=0.18.4
- **Cobblemon**: Required
- **Farmer's Delight**: Required
